## English

A beautifully designed, highly interactive, and offline-capable Timer, Stopwatch, and Clock application built with modern web technologies. This application is optimized for both desktop and mobile devices, providing a seamless, "native-app-like" user experience.

### ✨ Features

*   **📱 Progressive Web App (PWA)**: Installable on iOS and Android. Works completely offline after the first load.
*   **🕰️ Clock & Alarm**: A large, easy-to-read digital clock with a built-in alarm feature that triggers visual alerts.
*   **⏳ Timer**: Includes quick presets (5m, 15m, 30m) and customizable minutes, featuring a sleek circular SVG progress bar.
*   **⏱️ Stopwatch**: High-precision stopwatch (accurate to centiseconds) with a lap recording function.
*   **🌙 Dark Mode First**: Beautiful dark mode by default, with a manual toggle for light mode in the settings.
*   **👆 Mobile UX Optimized**: Includes haptic-like touch feedback (button scaling), safe-area padding for notched phones, and disabled pull-to-refresh to make it feel like a truly native mobile app.

### 🛠️ Built With

*   **[React 19](https://react.dev/)** - UI Library
*   **[TypeScript](https://www.typescriptlang.org/)** - Static Type Checking
*   **[Vite](https://vitejs.dev/)** - Next Generation Frontend Tooling
*   **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first CSS framework for styling
*   **[Framer Motion](https://www.framer.com/motion/)** - Fluid animations and page transitions
*   **[React Router](https://reactrouter.com/)** - Client-side routing
*   **[Vite PWA Plugin](https://vite-pwa-org.netlify.app/)** - Zero-config PWA Framework

### 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yo-yo0613/Timer.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Timer
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

### 🎨 UI / UX Highlights

*   **Glassmorphism**: Subtle translucent backgrounds and blurs used on the navigation bar.
*   **Micro-interactions**: Buttons have an `active:scale-95` CSS property to mimic physical button presses.
*   **Fluid Transitions**: Pages transition smoothly using Framer Motion.
*   **Accessibility**: High contrast colors and semantic HTML structure.

## 繁體中文

這是一個設計精美、高度互動且支援離線使用的計時器、碼表與時鐘應用程式。採用現代網頁技術開發，針對桌機與手機設備進行了深度最佳化，提供極度滑順、類似「原生手機 App」的使用體驗。

### ✨ 特色功能

*   **📱 漸進式網頁應用程式 (PWA)**：支援安裝於 iOS 與 Android 手機桌面。首次載入後即可完全離線使用。
*   **🕰️ 時鐘與鬧鐘**：巨大且易讀的數位時鐘，內建鬧鐘功能，時間到時會觸發視覺警報。
*   **⏳ 計時器**：提供快速預設選項（5分鐘、15分鐘、30分鐘）以及自訂分鐘數功能，並搭配優雅的圓形 SVG 進度條。
*   **⏱️ 碼表**：高精度碼表（精準至百分之一秒），並支援計圈 (Lap) 功能。
*   **🌙 深色模式**：預設為極具質感的深色模式，也可在設定中手動切換為淺色模式。
*   **👆 完美的手機體驗 (Mobile UX)**：加入類似震動反饋的按壓縮小動畫、支援 iPhone 瀏海/底部橫條的安全區 (Safe-area) 避讓，並停用網頁預設的下拉重新整理，帶來真正的原生 App 體驗。

### 🛠️ 使用技術

*   **[React 19](https://react.dev/)** - UI 前端框架
*   **[TypeScript](https://www.typescriptlang.org/)** - 靜態型別檢查
*   **[Vite](https://vitejs.dev/)** - 快速的新一代前端建置工具
*   **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first CSS 樣式框架
*   **[Framer Motion](https://www.framer.com/motion/)** - 流暢的過場與元件動畫
*   **[React Router](https://reactrouter.com/)** - 客戶端路由管理
*   **[Vite PWA Plugin](https://vite-pwa-org.netlify.app/)** - PWA 離線支援與設定

### 🎨 介面與體驗設計亮點

*   **毛玻璃特效 (Glassmorphism)**：導航列使用了半透明模糊的現代設計風格。
*   **微互動 (Micro-interactions)**：所有按鈕皆配置了 `active:scale-95`，完美模擬實體按鈕被按壓時的回饋感。
*   **流暢過場 (Fluid Transitions)**：頁面切換時使用 Framer Motion 進行平滑過場。
*   **無障礙 (Accessibility)**：採用高對比度色彩與語意化的 HTML 結構。
